# GameXplore-Game-Management-System

**Backrgound**

The Game Management System is a Java-Based games library platform, allowing players to access their accounts, purchase games and view 
their scores and playtime. It aims to create a robust interface where developers can upload games, and players can browse, purchase, and manage their gaming experience. This system streamline the process of managing game libraries for both players and developers. Key features include game management, account creation, profile updates, score tracking, and developer tools for uploading new games. Business goals will center around creating a user-friendly platform with secure login mechanisms, effective data handling, and expandable libraries.

Originally developed in (2024-2025 / 1st Term) as part of an academic course, and uploaded to GitHub later for documentation and portfolio purposes.

**System Features**

1. Allows users to interact with the system based on their roles: Developer or Player.
2. Facilitates seamless interaction for both developers and players through role-based menus and functionalities.
3. Enables developers to create, update, display, and delete games from a shared game list.
4. Allows players to manage a personalized save list of games for easy access and tracking.
5. Provide search functionality and comprehensive game details for enhanced user experience.
