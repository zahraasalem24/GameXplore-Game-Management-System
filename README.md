# GameXplore-Game-Management-System

This project is a game management system designed to manage and explore various types of 
games. It allows users to interact with the system based on their roles: Developer or Player.

Role Management:
Facilitate seamless interaction for both developers and players through role-based menus and functionalities.

Game Management:
Enable developers to create, update, display, and delete games from a shared game list.

Pesronalized Save Lists:
Allow players to manage a personalized save list of games for easy access and tracking.

Interactive Search and Display:
Provide search functionality and comprehensive game details for enhanced user experience.
